## 他是谁？

**Simon Verstraete**（又名 Simon Houdini / Siver），SideFX 官方 Tech Artist，曾任 **Project Titan**（SideFX × UE5 技术演示项目）的 **Project Lead**，现为 SideFX 核心教程讲师，内容涵盖 Houdini × UE5 游戏工具开发、PCG、VFX、Houdini Engine。

## 五、PCG 方向匹配度总览

| 教程 | 匹配度 | 理由 |
|---|---|---|
| **Building Generator** | ⭐⭐⭐⭐⭐ | 城市/建筑 PCG 核心，必看 |

| **BSP Building in UE** | ⭐⭐⭐⭐⭐ | UE笔刷→建筑，Houdini Engine 实时交互 |
| **Tree Generator** | ⭐⭐⭐⭐ | 植被 PCG，简历需要补强的方向 |
| **Rock Spawner** | ⭐⭐⭐⭐ | 场景物体撒点，通用 PCG 技能 |
| **Project Titan Cables** | ⭐⭐⭐ | 管线思维有价值，内容偏专项 |
| **Voxyde VFX系列** | ⭐⭐ | 影视向，锦上添花 |

## 六、推荐学习路径

```
阶段1：PCG核心（必看，2-3天）
  ├─ Games Workshop - Building Generator（35min）
  ├─ BSP Building in UE4（30min）
  └─ WFC Dungeon Generator（1h）

阶段2：扩展技能（选看，1-2天）
  ├─ Tree Generator（1h）
  └─ Rock Spawner Tool（30min）

阶段3：管线思维（进阶，2-3天）
  ├─ Project Titan Cables 6集（了解完整 HDA→UE5 管线）
  └─ 翻阅 YouTube 频道其他 Houdini Engine 相关视频

阶段4：VFX补充（按需）
  └─ 有选择地看 Voxyde 系列
```
