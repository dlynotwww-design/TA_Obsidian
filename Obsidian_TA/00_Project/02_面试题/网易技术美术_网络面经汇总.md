# 网易游戏技术美术 — 网络面经汇总

> 整理自牛客网、知乎、小红书、CSDN、看准网等多个平台的网易游戏技术美术（TA）面试经验。
> 覆盖网易互娱、网易雷火两大事业群，时间跨度为 2021-2025 年。

---

## 一、网易互娱 TA — 完整面经记录

### 📌 面试流程

```
投简历 → 筛选 → 笔试/测试题 → 一面(技术) → 二面(技术交叉) → HR面 → Offer
```

- **笔试形式**：网易互娱 TA **不考算法**，主要是**测试题（做作品）**——给一个命题，要求在一定时间内完成并提交
- **面试重点**：围绕测试题/作品集深入追问，图形学**不一定会问很多**（看面试官风格）
- **面试轮次**：通常 2 轮技术面 + 1 轮 HR 面

### 📌 互娱 TA 一面真题（2021 校招）

| # | 问题 | 考察方向 |
|---|------|----------|
| 1 | 模型法线从世界坐标系转换到相机坐标系用什么矩阵？ | 图形学/数学 |
| 2 | 这个变换矩阵的形状（Matrix Shape）？ | 线性代数 |
| 3 | 采样时 UV 如果在 [0,1] 之外怎么办？ | 贴图/纹理 |
| 4 | 延迟渲染的优缺点 | 渲染管线 |
| 5 | 对 HDR 的了解 | 颜色空间 |
| 6 | 对伽马空间和线性空间的了解 | 颜色科学 |
| 7 | 对 IBL（基于图像的照明）的了解 | PBR/光照 |
| 8 | 对光照贴图（Lightmap）的了解 | 烘焙/性能 |
| 9 | 各向异性滤波和 Mipmap 分别用来解决什么问题？ | 纹理采样 |
| 10 | 描边有哪几种做法？ | NPR/风格化 |
| 11 | 最近在玩什么游戏？ | 游戏感知 |
| 12 | 游戏里的液体是怎么实现的？ | 特效/Shader |
| 13 | 讲讲自己简历上的项目 | 项目经验 |
| 14 | 简历项目中的技术如何运用到商业游戏中去？ | 落地思维 |
| 15 | 身边朋友评价你是一个怎样的人？ | 软技能 |

### 📌 互娱 TA 实习面经（2025 届）

根据 2025 届拿到互娱 TA 意向书的同学分享：
> "面试阶段，网易就问了我测试题的细节问题，图形学也没怎么问"

- **核心**：测试题/作品集是网易互娱 TA 面试的绝对重点
- **流程**：一面二面连续，HR 面在外地同学可以选择线上
- **时间线**：笔试后可能 2 周以上才发面试通知（节奏偏慢）

## 二、网易雷火 TA — 完整面经记录

### 📌 面试流程

```
投简历 → 笔试/测试 → 一面(技术基础) → 二面(综合深挖) → HR面
```

- **雷火特点**：难度高于互娱，有测试题，面试涉及**渲染 + 大量 C++ + 计算机基础**，可能设压力面
- **测试题**：部分岗位要求 7 天内完成 3D 场景测试，考察造型/材质质感/UV规划/面数控制
- **二面**：典型的技术交叉面 + 总监综合面

## 四、网易 TA 面试官建议

> 来源：牛客网 — 网易雷火 TA 面试官分享

### 🎯 面试官最看重的三个能力

1. **图形学基础扎实**：渲染管线、PBR、光照模型、阴影技术——这些都是"一开口就知道你几斤几两"的硬功夫
2. **有实际项目**：不要只说"我学过"，要说"我做过"。Demo/作品集是最有力的证明
3. **学习能力和热情**：技术美术面很广，没人能全懂。但面试官能判断出你是"真的想学"还是"背了答案"

### 🎯 常见减分项

- 说"PBR 就是基于物理的渲染"——太浅，等于没说
- 技术名词堆砌但不能深入解释任何一个——暴露"只会用不会懂"
- 作品集里放了别人的东西或自己没深入做的——被追问时直接暴露
- 不懂装懂——面试官一眼能看出来，比坦诚说"这块我不熟"扣分多得多

### 🎯 雷火 TA 面试官原话

> "我们最怕的是那种简历上写了一堆技术名词，但问深一点就卡壳的。图形学基础不牢的，其他再花哨我们也不放心。"

## 五、网易 TA 测试题（笔试）特征

### 📌 互娱测试题

- **形式**：给一个命题（如"做一个风格化场景"），要求在规定时间内（通常 5-7 天）完成并提交
- **考察维度**：
  - 造型能力（模型质量）
  - 材质/贴图质感（PBR 流程熟练度）
  - UV 规划合理性
  - 面数控制与性能意识
  - 整体美术效果
- **提交物**：引擎工程文件（UE/Unity） + 截图 + 制作说明

### 📌 雷火测试题

- **难度**：普遍认为比互娱更难
- **形式**：可能包含 3D 场景制作 + 技术方案设计
- **额外考察**：对渲染效果的理解、工具链意识、文档能力
