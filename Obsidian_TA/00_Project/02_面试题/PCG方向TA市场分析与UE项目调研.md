## 一、PCG方向TA 市场招聘要求（2025-2026）

### 1.1 硬技能栈

#### 第一梯队：必须精通

| 技能 | 具体要求 |
|---|---|
| **Houdini** | 程序化建模 + VEX脚本 + HDA封装 + Houdini Engine集成 |
| **UE5 PCG Framework** | PCG Graph、关卡实例散布、World Partition |
| **Python** | 管线工具、批量处理、DCC自动化 |
| **Blueprint** | 自定义PCG节点、工具开发、快速原型 |

#### 第二梯队：强烈加分

| 技能 | 为什么重要 |
|---|---|
| **C++** | 腾讯/网易高级岗明确要求，UE引擎插件开发必备 |
| **PDG 分布式** | 大规模资产批处理，大厂管线标配 |
| **Runtime PCG** | 2025年需求爆发，引擎内运行时生成 |
| **HLSL/Shader** | 支撑程序化环境的材质和视觉效果 |
| **性能分析工具** | RenderDoc、Unreal Insights、GPU Profiling |

#### 第三梯队：2025-2026 新趋势 🤖

| 新技能 | 说明 |
|---|---|
| **AI + PCG 融合** | 腾讯明确要求 AI Agent 工作流 + PCG管线结合 |
| **AI辅助编程** | 用 GPT/Claude 提升工具开发效率 |
| **USD 管线** | 大厂开始推 Universal Scene Description |
| **移动端优化** | 手游大世界项目增多，移动端PCG成为新蓝海 |

### 1.2 不同级别要求

| 级别 | 年限 | 月薪参考 | 核心要求 |
|---|---|---|---|
| **初级 TA** | 1-3年 | 15K-30K | Houdini + Python + 一款引擎PCG落地 |
| **高级 TA** | 3-5年 | 30K-50K | 独立主导PCG管线 + C++ + 跨部门协作 |
| **资深/专家** | 5年+ | 50K-70K+·16薪 | 全局管线设计 + Runtime PCG + 团队指导 |

### 1.3 我的简历 vs 市场要求 Gap

| 市场要求 | 简历现状 | 差距 |
|---|---|---|
| Houdini VEX | 只提了Houdini/HDA，没提VEX | ⚠️ 需补充 |
| C++ | 完全没有提及 | 🔴 **最大短板** |

| 作品集/Demo | 完全没有链接 | 🔴 **致命缺失** |
| 移动端经验 | 无 | ⚠️ 看岗位方向 |

### 3.3 核心发现

#### 1. 自然野外 = 绝对刚需（75% 项目需要）

PCG TA 的主战场。包含：
- 地形 + 生态分区（山脉/森林/草原/沙漠）
- 植被散布与密度控制
- 道路/河流网络

#### 2. 都市场景正在爆发（33%，且全是重点项目）

| 项目 | 风格 |
|---|---|
| 异环 | 现代都市（霓虹 + 超自然） |
| Varsapura | 写实现代都市 |
| 白银之城 | 维多利亚蒸汽朋克都市 |
| 七日世界 | 末日都市废墟 |
| 二杠三 | 二次元都市 |

#### 3. 古风/武侠是一个独特细分市场

致金庸 + 燕云十六声 + 永劫无间 + 湮灭之潮（中世纪）——这类项目对古建筑程序化生成有特殊需求，建筑学背景 + Houdini 建筑生成经验是独一无二的优势。

#### 4. 室内 PCG 需求少但供给更少

只有 2/24 的项目有明确的室内/地下城需求，但市面上能做室内 PCG 的 TA 极少——这是蓝海卖点。

## 四、技能-环境-项目匹配分析

### 4.1 我的技能 × 环境需求

| 我的技能 | 匹配环境类型 | 对标项目 | 匹配度 |
|---|---|---|---|
| 城市/建筑生成 | 都市、古风城镇 | 异环、Varsapura、白银之城、致金庸 | ✅ 强匹配 |
| 植被PCG | 自然野外 | 几乎所有项目 | ✅ 强匹配 |
| 室内PCG | 室内/地下城 | 微恐PVEVP、骤影绯月杀 | 🟢 蓝海差异化 |
| 道路系统 | 都市 + 自然过渡 | 开放世界通用 | ✅ |
| 藤蔓/特效 | 奇幻/异世界 | 源初之结、沙盒探索 | ⭐ 亮点 |
| ❌ 地形PCG | 自然野外 | 75%项目需要 | 🔴 最大缺口 |
| ❌ 水体生成 | 水域 | 3个项目明确需要 | ⚠️ 短板 |

| 公司 | 推荐度 | 理由 |
