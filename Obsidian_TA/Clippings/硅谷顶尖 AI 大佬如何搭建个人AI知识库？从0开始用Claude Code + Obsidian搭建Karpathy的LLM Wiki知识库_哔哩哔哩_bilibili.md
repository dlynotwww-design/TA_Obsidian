# 硅谷顶尖 AI 大佬如何搭建个人AI知识库？从0开始用Claude Code + Obsidian搭建Karpathy的LLM Wiki知识库

> 用 Claude Code + Obsidian 搭建硅谷顶尖AI大佬的 LLM Wiki 知识库，让 AI 帮你持续整理、学习和进化知识。

这套方法来自 **Karpathy** —— OpenAI 联合创始人，前特斯拉 AI Director。

## 核心流程

1. **知识摄取（ingest）** —— 将各种来源的内容导入知识库
2. **查询学习（query）** —— 通过 AI 对知识库进行问答和学习
3. **知识库巡检（lint）** —— 让 AI 自动检查和维护知识库的完整性
