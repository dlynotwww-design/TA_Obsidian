# Monument Valley

> ustwo games，2014

## 核心玩法

基于不可能几何（Escher 风格）的空间解谜。玩家旋转世界、移动角色穿过视觉矛盾的路径。

## 关键设计

- **等距视角** → 空间错觉成为谜题核心
- **极简 UI** → 无 HUD，全沉浸
- **色彩系统** → 每个章节有自己的调色板
- **叙事** → 无声、抽象、诗意

## 参考价值

- 空间解谜的设计范式
- 极简美学在游戏中的落地
- 关卡作为「景观」而非「障碍」
