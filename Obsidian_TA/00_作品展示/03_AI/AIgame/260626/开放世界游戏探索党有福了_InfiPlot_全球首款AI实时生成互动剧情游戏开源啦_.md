选来选去，其实都在编剧写好的5条线里打转。所谓的"自由"，不过是在一个精心设计的迷宫里找出口——而迷宫的墙，早就砌好了。

如果我说，有一种游戏，没有预设剧本、没有固定角色、没有写好的结局呢？

你说的每一句话，AI都在实时生成接下来的剧情、画面、甚至配音。不是那种"选个选项然后播一段预制动画"，而是真正的——你想到哪，它就到哪。

这个项目叫InfiPlot，来自清华大学和兰州大学的一群年轻人。用他们自己的话说： "一款用AI实时生成内容的《完蛋！我被美女包围了！》"

# 先说体验：你到底在玩什么

打开InfiPlot，没有角色创建界面，没有世界观说明书。你只需要用一句话描述你想要的故事——

"穿越到哈利波特世界，我是斯莱特林的新生。"

"回到唐朝，我是一个赶考的书生。"

"在赛博朋克2077的世界里，我是个赏金猎人。"

然后，游戏开始了。

AI会根据你的描述，实时生成角色、场景、剧情。不是提前画好的立绘，是真的用AI现场画——每一幕场景的背景图，都是根据当前剧情实时渲染的。

画面风格也不固定。你可以让它画日漫风、写实风、水墨风、甚至"方格纸上的火柴人"——每次游玩的UI都会契合当前故事，而不是一成不变的模板。

更绝的是交互方式。你不需要点那些"选项A/B/C"的按钮——直接点击画面本身就行。

点人物？触发一段对话。点远处的门？可能推进剧情。点桌上的东西？可能只是插入一段场景描写，不换画面。

这背后是一个视觉大模型（VLM）在实时解读你的点击意图：你是在"探索当前场景"还是"要去新地方"，它分得清。

# 技术架构：一个AI"剧组"在后台运转

能做到"无剧本却剧情连贯"，靠的不是一个万能大模型，而是一个多智能体协作框架——像真正的影视剧组一样分工。

InfiPlot把AI拆成了五个职能角色：

| Agent | 职责说明 |
| ---- | ---- |
| 架构师(Architect) | 负责把控全局,制定宏观故事大纲,确保长线剧情不崩坏。 |
| 编剧 (Screenwriter) | 填充具体情节,编写细腻的对话文本与互动选项。 |
| 角色设计师 (Character Designer) | 锁定NPC的人设与外貌特征,解决AI绘画中常见的"变脸"难题。 |
| 场景布置师 (Scene Setter) | 将文字转化为精准的图像提示词(Prompt)，定义构图与光影。 |
| 画家 (Painter) | 调用文生图模型,实时渲染出电影级的场景插画。 |

这五个Agent相互配合，确保剧情连贯、角色不"变脸"、场景不穿帮。

还有一个关键设计：预测式渲染。

你在阅读当前场景时，引擎已经在后台悄悄生成你可能选择的下一个分支场景。等你真正选定方向时，画面通常已经画好了——切换瞬间完成，几乎没有等待感。

目前的延迟大约在10秒左右，团队的目标是压缩到5秒以内。

# 部署：Vercel一键，或者Docker自托管

InfiPlot支持多种部署方式。最省事的是Vercel一键部署，填好API Key就能跑。

需要接的模型供应商有四类：

| 类型 | 推荐方案 | 说明 |
| ---- | ---- | ---- |
| 文本（剧情） | DeepSeek V4 Flash | 成本极低 |
| 图像（场景） | Runware FLUX.2 Klein 9B | 每张约$0.00078 |
| 视觉（点击解读） | Gemini 3.5 Flash | 识别点击意图 |
| 语音（角色配音） | 小米MiMo TTS（免费）或阶跃星辰 | 可选，留空静音运行 |

也可以Docker自托管，支持x86和ARM（包括Apple Silicon Mac），两条命令搞定。

成本方面，每幕场景的主要开销在图像生成——FLUX.2 Klein一张图不到一美分，文本模型用DeepSeek V4 Flash几乎可以忽略。引擎会预测式生成你可能选的分支，所以实际花费会比看到的场景数略高一点，但整体还是很便宜的。

# 团队与愿景

InfiPlot的团队来自清华、兰大等高校，都是一群既玩游戏又懂技术的年轻人。

他们自己就是galgame、乙女游戏、FMV、AI角色扮演游戏的深度用户。在项目文档里写得很直白： "如果能选择不被预设的剧情选项，或者和AI角色深度互动而不只是通过聊天软件聊天，该是多么愉快刺激的体验。"

项目灵感来自2026年4月发布的flipbook——一种全新的AI交互式叙事形态。他们决定做一款类似的产品，但更强调"无边界"——没有预设剧本，完全由玩家驱动。

目前项目还很早期，已经实现了：延迟优化至约10秒、视觉识图交互、一键部署、前端直配API Key、移动端适配、剧情分享（.infiplot格式）。

还在路上的有：移动端App、ComfyUI自定义生图、5秒以内延迟、剧情存档续玩、自定义角色卡与世界观。

# 一些想法

InfiPlot让我想到一件事：AI游戏和AI聊天机器人，本质上可能是同一个东西。

CharAI、ChatGPT角色扮演、各种AI伴侣App——它们的核心都是"和一个AI角色聊天"。但InfiPlot把这个体验拉进了一个更有沉浸感的框架：有画面、有场景、有音效、有"世界"。

不再是"对话框里打字"，而是"在一个活生生的故事里做选择"。

技术上，多智能体协作 + 预测式渲染 + 视觉交互这条路线，是目前我见过的最聪明的AI游戏架构之一。它没有试图训练一个"什么都能做"的巨型模型，而是把任务拆解给多个专职Agent，每个Agent只需要做好一件事。

开源协议是AGPL-3.0，代码全在GitHub上。如果你也想做一个"属于自己的互动故事"，现在就能玩。
