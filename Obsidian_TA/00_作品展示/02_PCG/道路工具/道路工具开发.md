# 🛣️ 道路工具开发

> **项目类型**：UE PCG 道路系统工具
> **技术栈**：Houdini + UE5
> **技术难度**：⭐⭐⭐⭐

---

## 一、背景与需求

需求：样条线拉出街道，街道两边能添加装饰物如：消防栓 行道树 自行车 防护栏 垃圾桶 电箱 电线杆 路灯等，需要有资产替换 模型缩放 旋转 间距 数量 随机等功能，样条线交互处能自动生成十字路口，十字路口有红绿灯等相关元素

## 二、技术方案

### 2.1 整体功能

```
输入：Spline 路径
输出：完整道路（路面 + 标线 + 路肩 + 行道树 + 路灯 + 消防栓）

功能列表：
├── 多宽度支持（主干道 12m / 次干道 8m / 支路 5m）
├── 自动标线（虚线/实线按道路等级）
├── 行道树自动间隔散布
├── 路灯自动排布（偏移 1.5m，间隔 30m）
├── 消防栓/垃圾桶等街道家具
└── 交叉口自动处理
```
