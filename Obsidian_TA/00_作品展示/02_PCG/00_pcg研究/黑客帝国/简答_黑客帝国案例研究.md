# 黑客帝国：觉醒 — 城市生成管线简述

## 一句话总结

> 用 Houdini 做完整城市布局+建筑的程序化生成，导出点云，再用 UE5 的 Rule Processor 把点云重建为 Nanite 实例化城市，最后用 Mass AI 系统赋予 3.5 万行人和 1.8 万车辆的生命力。

---

## 核心流程

```
  道路骨架 → 地块划分 → 建筑填充 → 点云导出 → UE5 引擎重建 → Mass AI 注入生命
```

---

## 分步拆解

### 第一步：城市骨架 — City Layout（Houdini）

**输入：** 两条样条线（城市边界 + 主干道方向）

**做了什么：**
- 最短路径算法生成主干道网络
- 自动匹配网格模式，生成美式街区布局
- 输出：路网 / 地块 / 人行道 / 交通密度 / 行人密度 等元数据

### 第二步：道路生成 — Road & Freeway

**普通道路：** 三级层级（20m/10m/5m 模块），9 种基础砖块拼接，导出为实例化点云

**高速公路：** 不一样——因为是追逐场景，需要有机曲线，所以用 Mega Mesh 整体生成后再切成 100m 块

### 第三步：地块与建筑 — Building DNA

**地块处理：** 抠掉高速路区域 → 几何清理 → 按建筑高度和可用面积细分

**Building DNA 系统（核心创意）：**

```
每个建筑 = FootPrint（17种横截面） + 高度（底层/中间层/屋顶三层） + BDF（建筑风格）
```

- 建筑立面由 **Shape Grammar**（一种类似正则表达式的语法）控制模块排列
- 相邻建筑被遮挡的墙面自动剔除，减少 **25%** 模块
- 屋顶用 **波函数坍缩（WFC）** 算法：33 个可平铺网格 + 5,000+ 约束 → 生成 2,600+ 独特屋顶

### 第四步：细节填充 — Set Dressing

| 层级 | 内容 |
|------|------|
| 地面 (Ground) | 250 万块预位移地砖（每块 50 万面）→ 共 1.25 万亿三角形 |
| 人行道 (Sidewalk) | 样条驱动的模块散布 |
| 生态群落 (Biomes) | 预组装资产包（商场/停车场/高速路下方），用 Packing Scatterer 避免重叠 |
| 街道家具 (Furniture) | 路灯/消防栓/公交站/交通灯等，程序化散布 |
| 道路贴花 (Decals) | 车道线/斑马线/箭头/井盖，投射匹配道路几何 |
| 音频 | 3m 密度的点云，记录混响/音效类型 |

### 第五步：点云导出

Houdini 最终不是导出 Mesh，而是导出 **PBC（Point-Based Cache）** —— 只包含位置+属性的点云。

这很关键：**DCC 和引擎解耦**，点云可以在 UE5 中灵活决定如何实例化。

### 第六步：UE5 引擎重建 — Rule Processor

**Rule Processor**（项目插件，非引擎内置）做三件事：

1. 读取 PBC 点云
2. 按规则把每个点替换为对应资产（建筑/道路/贴花/声音）
3. 智能回收：追踪 Actor，自动清理旧规则生成的内容

重建后的城市用 **World Partition + HLOD 三级策略** 管理加载：
- 128m 内：完整资产
- 128m-768m：合并 ISM 的 HLOD0
- >768m：单 Nanite 网格的 HLOD1（远景）

### 第七步：注入生命 — Mass AI

```
ZoneGraph（轻量级廊道替代 NavMesh）
    ↓
MassEntity Framework（数据导向 ECS — Fragment + Processor）
    ↓
┌──────────────┴──────────────┐
│   Crowd (35,000人)        Traffic (18,000辆)   │
│   StateTree 状态机         车道队列/变道/并线      │
│   4 级 LOD (MetaHuman↔顶点动画)   4 级 LOD (物理车↔ISM) │
│   力基避让 + 持久性          交叉口协调器            │
└──────────────┴──────────────┘
```

## 面试时可以强调的三个亮点

1. **量化世界思想** — 不用 OSM 真实数据，而是自建模块体系 + 实例化，换来完全的几何控制和可预测性

2. **点云作为中间表示** — Houdini 不直接输出 Mesh，只输出点云+属性，DCC 和引擎完全解耦，Houdini / Blender / 任何工具都能接入

3. **程序化+艺术控制的平衡** — Building DNA（参数化多样性）+ Shape Grammar（美术可控的立面）+ WFC（33 网格→2,600+ 屋顶）+ Volume Override（后期锁定调整），不是"全自动黑箱"，而是"参数化的美术工具"
