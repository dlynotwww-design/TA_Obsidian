### 总结：近十年 PCG 的技术进化轨迹

| **演进维度** | **10年前（2016左右）**      | **5年前（2021左右）**   | **2025-2026 现状**              |
| -------- | --------------------- | ----------------- | ----------------------------- |
| **主要工具** | 独立引擎底层代码 / 基础 Houdini | 高阶 Houdini / 引擎插件 | **UE5 内置 PCG** / 实时 GPU 节点    |
| **生成时机** | 100% 离线生成（烘焙进游戏包）     | 离线生成为主 + 部分运行时拼砖  | **完全运行时（Runtime）实时计算生成**      |
| **美术控制** | 比较死板，容易穿模或千篇一律        | 美术绘制 Mask，控制力大幅提升 | 美术与程序员在同一张**逻辑图表**上协同         |
| **核心算法** | 噪声函数 / 细胞自动机 / 汪氏砖    | 物理侵蚀模拟 / 语法文法树    | 空间采样（Spatial Sampling） / 语义理解 |
