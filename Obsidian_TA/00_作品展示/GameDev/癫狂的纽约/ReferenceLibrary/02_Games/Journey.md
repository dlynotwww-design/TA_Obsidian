# Journey

> thatgamecompany，2012

## 核心玩法

无文字的沙漠朝圣之旅。随机的匿名多人。爬山。

## 关键设计

- **「滑沙」机制** → 纯粹的移动快感
- **匿名多人** → 无法对话，只能吟唱
- **围巾长度 = 飞行能量** → 视觉化的能力升级
- **色调叙事** → 沙漠→海底→雪山的色彩弧线

## 参考价值

- 如何用关卡节奏讲一个没有台词的故事
- 情感曲线设计：高潮→平静→绝望→升华
- 空间尺度的控制：宏大沙漠 → 压迫隧道 → 无尽雪山
