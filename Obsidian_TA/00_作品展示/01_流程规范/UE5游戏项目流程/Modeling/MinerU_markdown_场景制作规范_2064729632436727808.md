## 贴图输出

贴图格式Tga 通道样式处理

1. 颜色贴图后缀 d 颜色贴图Alpha 用作透贴或金属性（Metallic）两者不可同时存在，

2. 法线贴图后缀_n 法线蓝通道放置Roughness, Alpha 放置AO

3.自发光贴图后缀_e 自发光颜色贴图无Alpha

建议创建2个导出格式预设，一个无Alpha,一个有Alpha。命名：M5 和 M5_A。自发光贴图输出根据情况调整。

## a. 模型

01. 确认MAX单位设置正确（依据项目相关要求检查）

02. 确认模型尺寸比例（参考提供的基础模型及人物大小比例正确）

03. 确认模型法线方向正确, normal贴图方向正确 (Max烘焙normal需翻转)

04. 确认对应的点；线；面无误（禁止出现模型点未合并，同一个面禁止超过4个顶点，禁止出现镂空面重合面）

05. 光滑组分配合理（以Max光滑组为准，禁止使用Maya软硬边）

06. 正确设置物件坐标，且坐标归为0点。一般以物体中心底为准，部分场景根据物件自身情况自行判断合理即可。

07. 模型FBX导出正确.清理无用模型信息，即 ResetXForm。模型导出时，关闭不需要的关联信息，如：动画，摄像机，灯光...）

## b.UV

01.UV1边界距离合理（一般UV与UV间保持8个以上像素）

02. UV2合理使用避免浪费。分配中避免重叠，能合并的UV尽可能的合并。2U的接缝尽可能的隐藏到模形背面。模形底面；背面或者不重要的面，UV尽可能的小。UV与UV间保持一定距离，不要太近。

## 概述及流程

产品定位为手游产品，同时还预备了PC端。在品质质量上会要求比较高。外包制作时，需要制作人员手游资源制作上有一定了解，要能区分处手游和端游区别。端游出于平台自身性能强大，在模形和贴图制作时基本可以高面数大尺寸贴图；手游处于手机自身的性能局限性，所有要求在模形和贴图在制作时需要更加合理的分配资源。

手游制作基本要求：模形上需要保证整体剪影效果和大形体结构，小结构小细节全部概括或者不要烘培在面上；重复或者类似的模形，同时配合UV分配尽可能共用；根据制作物件的大小和重要重要不同，具体问题具体分享，制作时列表会详细描述。贴图上以写实绘制为主，按正常PBR次世代流程制作。制作时注意材质区分和质感表现；小细碎的细节不需要；高密度的纹理需要放大。在手机屏幕上细碎的细节基本看不到，高密度的纹理显示会闪屏，制作时需要注意。制作植物插片时，需要按组制作，单片制作不允许。透贴和金属不允许同时出现。

## 制作流程(制作是根据实际情况选择相应顺序流程，需要灵活变通西靠，注意提交节点)

通用流程：中模 → 高模/卡线模 → 低模 → UV+烘焙+UV2 → 贴图 → LOD+碰撞 → Uniyt效果 → 提交

建筑流程：基础白模(提供) → 贴图规划基础模型初贴 → 中模(包括模块拆分) → 高模/卡线模 → 低模 → UV+烘焙+UV2 → 贴图 → LOD+碰撞 → Uniyt效果 → 提交

植物流程：中模 → 高模 → 低模 → UV + 烘焙 + UV2 → 贴图 → 低模 / LOD → + 碰撞 → Uniyt 效果 → 提交 (除了树木树干需要雕刻以外，草木灌木类和植物叶子不需要制作高模，直接贴图然后底面插片)

1. 低模制作，剪影优原则，尤其是自然资源类，其他类别也需稍加注意。

2.应用于Unity引擎，场景贴图Normal方向对应Maya格式。

3. 场景制作提交以Max为准，包括模型组，光滑组等。（提供基础比例文件）

4. 粗糙度格式为roughness（越亮越粗糙，越暗越光滑）

5.基础白模只是确定该物件整体大小，其他信息请根据原画和制作要求进行。

6. 每阶段需要确认方可进入下一步。

7.从中模提交开始，每个阶段都要自动展2UV，最终提交需要仔细按照要求展2UV。

8.从中模开始必须严格按照规范命名、建立正确的层级、材质球命名正确、FBX正确导出。

9.从烘焙法线阶段开始，比如在引擎查看效果并且截图，最终贴图还需要在引擎烘焙灯光贴图（2UV），根据烘焙的效果调整贴图的饱和度明度，然后在引擎截图并提交。

注：基础白模，根据实际情况提供；若未提供，请根据制作原画和制作要求完成中模。

## 室内物件-需要增加Alpha顶点色

注意事项和要求：一般情况根据原画或者照片制作，整体制作时没有特殊要求的话，一般都是8成新左右，需要注意形体结构，植物盆栽类还需要注意剪影。

高模阶段制作时，规则物件卡线制作雕刻局部结构细节即可；不规则物体如石雕假山等根据要求雕刻。

低模阶段，形体需要概括，不要浪费面数。所有小结构能烘培在平面上的全部优化，只需要保证大结构即可。

陶瓷罐子类 圆柱形结构，单个物件控制在50-200面以内，组件控制在500以内。小物件一般高度50cm以下，使用6段-8段圆柱结构，杯子口内部延申并收缩到一点。

生活类物件 1. 所有镂空结构避免；编织的竹制纹理的结构制作比现实的需要放大制作（至少放大一倍），编织太密太小手机屏幕基本看不到还很糊。

2.两个大的面之间接近垂直，需要增加一个倒角。如箱子连个侧面；枕头的顶面与侧面。

3. 圆形的编织结构，如竹篮，圆形使用12-16段圆编织。篮框里面面熟上适当优化。

UV1:在空间环境下，无论物体怎么旋转角度，人肉眼只能看到物体的三个面，大部分物件类型都在2米以下，在手机屏幕上比较小。注意UV共用;根据物体的造型UV1/2左右共用;1/4共用;平面结构平展。

UV2:所有能合并的UV必须合并，然后平展。接缝尽可能的留在不显眼的地方。如背面；有遮挡的侧面等。

贴图:根据制作要求，制作写实效果贴图. 要求有使用的痕迹，经常触摸的地方粗糙度需要亮一点. 注意主次关系，颜色和明度上的区分。贴图层次

LOD:不需要

碰撞: 尽可能的包裹住低模即可，不需要太多细节。根据材质球属性对应材质信息通道。

植物盆栽1.盆栽类物件单个面数控制400-500面之间。

2.叶片使用透贴制作，叶片制作要求以组的形式制作，然后模形插片。不可以单片叶子插片制作。

3. 盆栽的花盆，圆形结构视情况使用6段-12段。其他结构是情况要求概括模形。

4. 盆栽植物插片，贴图按组制作。

## detail_家具(通用物件-室内)

白模：确定大小尺寸

中模：根据提供任务要求结合原画和照片制作。中模要求对物件的形体和结构把握准确。中模制作时，需要对后续的高模；低模；UV进行规划和分配。

高模：此阶段模形根据中模的规划，重复性的模块只做一个，后期共用。制作时需要注意当前物件的级别。物件分为低；中；高，三个级别。

低级家具：边角做些小破损，手游的模形破损的结构需要概括，不需要做细节，不需要做映射纹理。制作时，注意共用部分尽量避免蝴蝶纹和重复感。在雕刻平面的结构时，如桌面；台面等，四个角的平面雕刻的纹理特征性不要太明显，游戏视角的会显得比较生硬不自然。一般低级的物件整体成色在7成新以上。做旧的同时，雕刻又硬破损和磨损。，磨损的结构边缘偏圆弧形，区别于磕碰的感觉。

中级家具：基本是就边缘的小磨损，不会出现硬磕碰，只是在颜色上做区分或者在粗糙度上的区分。

高级家具：基本就是全新，现实中是啥样就做成啥样!这样的模形基本就是卡线高模。

注意事项：建筑类制作需整体考虑，重要环节在于对整套建筑的贴图规划，和建筑拆分。规划好贴图有助于资源的合理利用而不浪费。

architect 物件大于高于6米，此类型贴图不支持透贴。单个模型不能超过7000面，一个mesh对应一个材质球，如一个mesh超过7000面以上，则根据情况优化或者拆分2各以上mesh。贴图规划若一张贴图无法满足需求，则情况增减贴图。同一组模形最多出现3各子mesh，即a b c。自发光资源单独规划，即模形单独mesh，贴图单独规划一张。制作时视情况调整。

建筑流程：基础白模(提供) → 贴图规划 → 中模(包括模块拆分) → 高模/卡线模 → 低模 → UV+烘焙+UV2 → 贴图 → LOD+碰撞 → Uniyt效果 → 提交（提交理成视情况变化灵活调整）

基础模形：确定当前物件的大小和比例方向，部分带有结构指引。

中模：根据提供的基础模形，结合原画和照片搭建中模。与此同时制作贴图规划，并进行粗贴与模形用来验证美术效果。制作时视情况对建筑进行调整。验证完美术进行模块拆分。

高模：按拆分制作。要求：结构把握准确，形体的造型剪影美观，细节结构概括，小碎细节不需要。

低模/2U：模形形体需要概括制作，小结构不需要，尽可能的烘培在贴图上。2U：能合并的UV尽可能的合并。底面看不到的部分，UV缩放最小。较细的结构和较小的结构UV放大至少一倍以上；UV的接缝放在物体背后或者不常见的位置。UV共用合理，利用率高，UV摆放必须在第一象限。

贴图：写实制作。注意主次关系，颜色和明度上的区分，整体效果避免噪点和脏色混乱，需用色干净。贴图层次：基础底色；破损/干裂/风化/褪色等；污渍/脏渍等。贴图必须是正方形，TGA格式。

LOD：分级LOD0-LOD1-LOD2，模型面数每级减至少50%以上，以效果和最优化为准。（后续查看具体规则）

LOD0/LOD1 - 2U需保持一致，LOD2不需要2U。

碰撞：建筑碰撞按整体模形制作成一个封低模形，不可漏面，不需要太多细节。根据材质球属性对应材质信息通道。

美术效果验证：LOD0为当前视窗显示，间隔50米放置LOD1，在间隔50放置LOD2。整体查看时效果接近即可，提交文件时，需提交间隔50米对比效果截图。

## 制作思路和流程讲解：

1.外包任务开始前，先确定制作要求和原画是否齐全，先找参考。了解当前制作内容，分析制作方法，了解植物的生长方式逻辑；同时消化技术文和发包制作要求。

2. 制作开始时根据原画或者照片参考制作中模，多角度的把握剪影结构和叶片的疏密关系，中模阶段提交树木需要树干和叶片（叶片用颜色色块或临时照片直接插片，只要大感觉快色造型，没有任何限制只要效果）；同时矮小灌木插片也适用于树叶的方法。高模阶段只有树干需要雕刻，需仔细分析树种和必要的环境影响。低模阶段，主要是造型剪影和面数控制。贴图需要共用率高，避免浪费纹理清晰，法线结构合理。插片更讲究插片方式和插片的形状来弱化模型感，树叶大小和密度关系按真实的表现。

中模：树木类推荐使用speedtree/ZbrushZ球快速造型，中模阶段要求比例结构造型合理剪影丰富，达到原设及参考的造型效果。

高模：结构把握准确，形体的造型剪影美观，细节结构概括，小碎细节不需要，映射纹理需清晰表达。

低模/UV1：面数符合手游逻辑，精简布线；结构合理剪影丰富；3米以下面数可稍微增加保证部分细节；UV共用合理，利用率高，UV摆放必须在第一象限。

贴图：纹理清晰，法线结构合理，反射贴图细节丰富。整体效果避免噪点和脏色混乱，需用色干净。树叶和灌木叶片按组制作，不允许单根枝条制作。提高贴图利用率。树叶和草木灌木等叶片部分只做颜色贴图，个别情况按要求制作。

LOD面数和级别：6米以下植物包括树，灌木等不需要制作LOD。6米以上的树木，LOD1面数优化至少50%以上到最精简，需要保证美术效果与LOD0接近。LOD2截图插片制作：引擎截三视图在PS整合成一张512贴图，max插片并调整贴图效果与引擎LOD1效果大致接近。

2U: LOD0/LOD1,2U需保持一致。LOD2无需2U。要求：接缝位置隐藏合理，UV摆放整齐工整，允许拉伸，看不到的面2U尽可能缩小。UV间隔8个像素左右。提交时自行Unity烘培检查。美术效果验证：植物类-6米以上树木效果验证方式：LOD0为当前视窗显示，间隔50米放置LOD1，在间隔50放置LOD2。整体查看时效果接近即可，提交文件时，需提交间隔50米对比效果截图。

植物都是需要Alpha顶点色,使用方式参考文档规范篇
